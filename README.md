将中文转为拼音。  
不依赖任何三方库，几KB原生js实现，有良好的兼容性，不挑运行环境。

## 使用方式

````
<script src="https://res.zvo.cn/pinyin/pinyin.js"></script>
<script>
var str = pinyin.convert('你好')
alert(str);
</script>
````
